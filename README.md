Sabana Digital
Professional Livestock Management for the Heart of Córdoba.
Gestión Ganadera Profesional para el Corazón de Córdoba.

📋 Project Overview / Resumen del Proyecto
Sabana Digital is a robust Backend solution designed to help farmers in the Córdoba region digitize their traditional operations. This platform allows for strict control over animal health, births, deaths, and financial inventory.

Sabana Digital es una solución de Backend robusta diseñada para ayudar a los ganaderos de la región de Córdoba a digitalizar sus operaciones tradicionales. Esta plataforma permite un control estricto sobre la salud animal, partos, muertes e inventario financiero.

🚀 Key Features / Funcionalidades Clave
Livestock Inventory: Detailed tracking by breed, gender, and farm location.
Health & Life Ledger: Accurate records of births, vaccinations, and medical events.
Weight Monitoring: Historical tracking of animal growth and market value.
Multi-Farm Management: Handle multiple properties under a single user account.
🛠️ Tech Stack / Tecnologías
Framework: NestJS (Node.js)
Language: TypeScript
Database: PostgreSQL
Architecture: Modular & Scalable
🏗️ Data Architecture / Arquitectura de Datos
The system is built on 5 fundamental pillars / El sistema se basa en 5 pilares:

Users: Farm owners and administrators.
Farms: Real estate and land location in the Sabana region.
Livestock: Individual identification via tag numbers (chapetes).
Health Events: Births, deaths, and medical history logs.
Weight Logs: Historical performance and weight records.
💻 Getting Started / Cómo Empezar
Clone the repository: git clone <repo-url>
Install dependencies: npm install
Environment Variables: Set up your .env with PostgreSQL credentials.
Run the server: npm run start:dev
